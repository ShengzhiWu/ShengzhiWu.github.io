Audio Creator

Audio Creator has a graph node UI (similar to Grasshopper, Houdini, Substance Designer, TouchDesigner, etc.), which can help users quickly build the logic. For example, if the user wanted to generate a siren-like sound, he or she can place two nodes that generate sine waves, one with a frequency of several hundred Hz (the frequency of the siren's sound) and another (the envelope) with a frequency of around 1 Hz (the frequency at which the siren's loudness varies), and then multiply the two signals and get the result.

Unlike most software with node editor, Audio Creator allows loops. This means a signal generated by a set of nodes can be fed to itself cyclically. A typical application is to implement an integrator that can numerically solve a differential equation, which is how the last audio shown above was generated. This also implies the fact that Audio Creator is Turing complete. So, in principle, Audio Creator can generate almost anything you want.
